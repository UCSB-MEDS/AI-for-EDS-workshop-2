# AI for Envinronmental Data Science 
## Workshop 2: GitHub Copilot & Reproducible Python Projects

This repository contains materials for the 2nd day of the AI for EDS Workshop Series for the [Masters in Environmental Data Science (MEDS)](https://bren.ucsb.edu/masters-programs/master-environmental-data-science/academics-meds-program), at the Bren School of Environmental Science & Management at UC Santa Barbara.

For more information visit the [MEDS AI for Environmental Data Science Workshop Series website](https://ucsb-meds.github.io/AI-for-EDS/).


**Disclaimer:** The data in this project is fake and is only meant for demonstration purposes.
